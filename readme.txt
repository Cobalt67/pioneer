This application is the copy of old computer game "pioneer"
Programming language: Python
The interface language: none
it is free for usage and copying anywhere
You can click by Left mousebutton on rectangle fields
The project is closed
If there is no mines, then the field will be painted magenta
if there is one mine around,then the field will be painted green
if there is two mines around then the field will be painted orange
if there is one mine then the game is over and all the map will be painted
